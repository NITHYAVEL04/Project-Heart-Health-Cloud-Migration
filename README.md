# Project-Heart-Health-Cloud-Migration
Cloud Infrastructure, ML Deployment, and System Reliability.
The "2026 Modernization" Update
Originally developed as a B.Tech project in 2024, I have utilized my current upskilling period to migrate this application to AWS. This repo now serves as a demonstration of Infrastructure as a Service (IaaS) application hosting.

🛠 Infrastructure Stack
Provider: AWS (Amazon Web Services)

Host: EC2 t2.micro (Ubuntu 22.04 LTS)

Web Layer: Flask (WSGI)

Data Layer: MySQL

Engine: Multi-Layer Perceptron (Deep Learning)

📈 DevOps Implementation
Manual Provisioning: Handled EC2 instance creation, Elastic IP association, and Security Group configuration (Ports 22, 80, 443).

Deployment: Configured the Linux environment, installed system-level dependencies, and managed Python virtual environments for application stability.

Monitoring: Utilizing basic AWS CloudWatch metrics to monitor instance health and CPU utilization.
